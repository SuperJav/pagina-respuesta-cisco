15.¿Cómo se realiza un ataque de pitufo?

  • al enviar un gran número de paquetes para desbordar la memoria intermedia asignada del dispositivo de destino
  • enviando un gran número de solicitudes ICMP a direcciones de difusión dirigidas desde una dirección de origen falsificada en la misma red *
  • al enviar una gran cantidad de paquetes TCP SYN a un dispositivo de destino desde una dirección de origen falsificada
  • enviando una solicitud de eco en un paquete IP mayor que el tamaño máximo de paquete de 65,535 bytes
  • @Javi__Super

16.¿Cuál es una característica de un caballo de Troya en relación con la seguridad de la red?

  • El malware está contenido en un programa ejecutable aparentemente legítimo. *
  • Cantidades extremas de datos se envían a una interfaz de dispositivo de red particular.
  • Un diccionario electrónico se usa para obtener una contraseña que se usará para infiltrarse en un dispositivo de red clave.
  • Demasiada información se destina a un bloque de memoria particular, lo que hace que se afecten áreas de memoria adicionales.
  • @Javi__Super

Un caballo de Troya realiza operaciones maliciosas bajo la apariencia de un programa legítimo. Los ataques de denegación de servicio envían cantidades extremas de datos a un host particular o una interfaz de dispositivo de red. Los ataques de contraseña utilizan diccionarios electrónicos en un intento de aprender las contraseñas. Los ataques de desbordamiento de búfer explotan los búferes de memoria enviando demasiada información a un host para que el sistema deje de funcionar.

24.¿Qué es un barrido de ping?

  • una técnica de exploración que examina un rango de números de puerto TCP o UDP en un host para detectar servicios de escucha.
  • una aplicación de software que permite la captura de todos los paquetes de red que se envían a través de una LAN.
  • un protocolo de consulta y respuesta que identifica información sobre un dominio, incluidas las direcciones asignadas a ese dominio
  • una técnica de escaneo de red que indica los hosts en vivo en un rango de direcciones IP. *
  • @Javi__Super

Un barrido de ping es una herramienta que se utiliza durante un ataque de reconocimiento. Otras herramientas que pueden usarse durante este tipo de ataque incluyen un barrido de ping, escaneo de puertos o consulta de información de Internet. Un ataque de reconocimiento se utiliza para recopilar información sobre una red en particular, generalmente en preparación para otro tipo de ataque de red.
