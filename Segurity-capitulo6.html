11.Dos dispositivos que están conectados al mismo conmutador deben estar totalmente aislados entre sí. ¿Qué característica de seguridad del switch de Cisco proporcionará este aislamiento?

  • PVLAN Edge *
  • DTP
  • LAPSO
  • Guardia de BPDU
  • @Javi__Super

La función Edge de PVLAN no permite que un dispositivo vea el tráfico generado por otro dispositivo. Los puertos configurados con la función Edge de PVLAN también se conocen como puertos protegidos. La protección BPDU evita la conectividad no autorizada a un conmutador de Capa 2 con cable. SPAN es la duplicación de puertos para capturar datos de un puerto o VLAN y enviar esos datos a otro puerto. El DTP (Dynamic Trunking Protocol) se habilita automáticamente en algunos modelos de switch para crear una troncal si el dispositivo conectado está configurado para trunking. Cisco recomienda deshabilitar DTP como una mejor práctica.

18.¿Cuál es el comportamiento de un interruptor como resultado de un ataque exitoso de la tabla CAM?

  • El conmutador reenviará todos los marcos recibidos a todos los demás puertos. *
  • El interruptor caerá todos los marcos recibidos.
  • Las interfaces del switch pasarán al estado de error deshabilitado.
  • El interruptor se apagará.
  • @Javi__Super

Como resultado de un ataque a la tabla CAM, un conmutador puede quedarse sin recursos de memoria para almacenar direcciones MAC. Cuando esto sucede, no se pueden agregar nuevas direcciones MAC a la tabla CAM y el conmutador reenviará todas las tramas recibidas a todos los demás puertos. Esto permitiría a un atacante capturar todo el tráfico inundado por el switch.
